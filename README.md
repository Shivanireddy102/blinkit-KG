# 🧠 AI-Powered Enterprise Knowledge Graph Builder

## 📌 Overview
This project builds an AI-powered knowledge graph using Neo4j and Large Language Models (LLMs).  
It converts structured data (CSV) into a graph database and allows intelligent querying using natural language.

---

## 🚀 Features
- CSV Data Ingestion
- Automatic Knowledge Graph Creation
- LLM-Based Query Processing
- Natural Language to Cypher Conversion
- Interactive UI with Streamlit

---

## 🛠 Tech Stack
- Python
- Neo4j
- Streamlit
- LLM (Groq / OpenAI)
- LangChain

## ⚙️ How to Run

1. Clone the repository  
   git clone <https://github.com/Shivanireddy102/blinkit-KG>

2. Install dependencies  
   pip install -r requirements.txt

3. Create a `.env` file and add required API keys  

4. Run the application  
   streamlit run streamlit_app.py
